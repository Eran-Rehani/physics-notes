# Waves and Optics Summary

A LaTeX summary of waves and optics, including Fourier methods and diffraction theory.

## Topics Covered

- Wave equation and general solutions
- Fourier analysis applied to waves
- Interference and diffraction
- Fresnel and Fraunhofer diffraction
- Electromagnetic waves and optics fundamentals

## Files

| File | Description |
|------|-------------|
| `main.tex` | Main summary |
| `formula cheet sheet.tex` | Quick-reference formula sheet |
| `Fourier.tex` | Fourier series and transforms |
| `Fourier solution to the WE.tex` | Fourier solutions to the wave equation |
| `fresnel and frenhofer.tex` | Fresnel & Fraunhofer diffraction |

## Usage

```bash
pdflatex main.tex
```
