# Analytical Mechanics Cheat Sheet

A comprehensive LaTeX cheat sheet covering key topics in analytical mechanics.

## Topics Covered

- Lagrangian and Hamiltonian mechanics
- Euler angles and rigid body dynamics
- Small oscillations and normal modes
- Effective potential and orbital mechanics (elliptic orbits)
- Buckingham π theorem (dimensional analysis)
- The Rayleigh Dissipation Function
- Taylor series expansions

## Usage

Compile any `.tex` file with a LaTeX engine:

```bash
pdflatex "cheat sheet.tex"
```

Or use [Overleaf](https://overleaf.com) for online editing.
